# Eoraptor Exchange Website

## 技术栈
- Frontend: Next.js 14 + Tailwind CSS + next-intl (i18n)
- Backend: Node.js + Express + MongoDB
- Admin: Custom CMS

## 功能模块
1. 多语言切换（中文简体、英语）
2. 后台管理系统（Banner、文案、文档管理）
3. 文档下载（白皮书、操作手册）
4. 交易所链接（后台配置）

## 项目结构
```
eoraptor-website/
├── frontend/          # Next.js 前端
├── backend/           # Node.js 后端
├── docs/              # 文档资源
└── README.md
```
